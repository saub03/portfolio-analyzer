# 금융 정보 수집 자동화

## 개요
중요한 거시 경제 이벤트와 관련된 뉴스 정보를 자동으로 수집하고 AI가 분석합니다.   
또한 개인 금융 포트폴리오 비중 변화량, 수익률, MDD등 분석 정보, 종목별 주요 뉴스를 수집해서 요약된 리포트를 자동으로 메일로 전송하는 파이프라인을 구축합니다.

## 상세 설계
- 대상(Target): 
    - 경제 일정 정보: https://www.investing.com/economic-calendar
    - 뉴스, 공시 정보: https://finance.naver.com/
    - AI 분석: Google Gemini API
    - 자산 시세 및 환율 정보: FinanceDataReader, 빗썸 API
- 주요 라이브러리(기술 스택):
    - finance-datareader: 주식 정보와 환율 정보를 불러옵니다.
    - Requests, BeautifulSoup4: 금융 뉴스를 스크랩할 때 사용합니다.
    - Pandas, numpy, matplotlib, seaborn, scikit-learn: 데이터를 분석하고 시각화 할 때 사용합니다.
    - Smtplib, email, jinja2: 수집, 정리한 데이터를 이메일로 전송합니다.
    - google-generativeai: AI를 이용해 뉴스를 분석합니다.
- 데이터 흐름
    - Input: investing.com과 네이버 금융에서 시황과 뉴스를 불러옵니다. FinanaceDataReader와 빗썸에서 제공하는 API로 주식, 암호화폐 가격 정보를 수집합니다.
    - Output: AI 시황 분석, 종목별 관련 뉴스, 포트폴리오 성과 분석 정보 등을 포함한 HTML 형태로 이메일 발송합니다.
- 프로세스 설계
    - 1. 윈도우 스케줄러 기능을 통해 매일 정해진 시간에 스크립트를 자동 실행합니다.
    - 2. 데이터 크롤러와 API를 통해 데이터를 수집합니다.
    - 3. 데이터 분석 라이브러리를 사용해 누적 수익률, 가격 변동, 상관계수 등을 계산합니다.
    - 4. 수집된 뉴스 및 시황 데이터와 포트폴리오 정보를 AI API 프롬프트에 전달해 AI 분석 코멘트를 생성합니다.
    - 5. jinja2 이용해 HTML 템플릿을 생성하고 텍스트를 구성합니다.
    - 6. 사용자에게 이메일을 발송합니다.

## 개발 계획
- 1주차: 데이터를 수집하고 전처리하는 파이프라인을 제작합니다.
- 2주차: 스크립트 실행 시 데이터를 수집하고 사용자에게 이메일을 전송하는 전체적인 프로세스를   구현합니다.
- 3주차: 분석 기능을 구체화하고 결과를 HTML 형태로 구성합니다.
- 4주차: 프로세스 중 발생할 수 있는 예외나 오류를 점검하고 최종 테스트를 합니다.
